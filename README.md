# INFNOVA Academy - Frontend Engineering Challenge

A modern, responsive course platform built for the INFNOVA Academy internship. 

## 🚀 Live Demo

(https://infnova-internship-program.vercel.app/)


## 🛠️ Tech Stack
- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS v4
- **Routing:** React Router Dom
- **Icons:** Lucide React
- **Data:** INFNOVA Course API

## ✨ Key Features
- **Dynamic Fetching:** Uses a custom `useFetch` hook to handle API data, loading states, and error management.
- **Skeleton Loaders:** Implemented custom pulse animations to improve perceived performance.
- **Responsive Layout:** Mobile-first approach with a sticky sidebar for the Course Details view.
- **Search Functionality:** Real-time filtering of courses by title, instructor, or category.

## 📝 Approach & Reflections
- **Data Integrity: ** implemented an `onError` fallback to ensure a stable UI.
- **Architecture:** Organized components into a modular structure for maintainability.
